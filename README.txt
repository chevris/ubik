=== Ubik ===
Requires at least: WordPress 4.7
Tested up to: WordPress 4.9

== Description ==

Ubik is a multipurpose theme to build almost any type of website.

== Documentation ==

= Customizing Ubik =

To customize the Ubik theme you have to open the Customizer by accessing the Appearance menu -> Customizer.
